Subscription Management System🚀 

A simple React app to manage user subscriptions. Add users, assign plans, and cancel subscriptions.

✨Features

Add users with a plan (Basic, Premium, Pro), 
View all subscriptions, 
Cancel subscriptions. 

⚡ Tech Used

HTML, CSS, React.js (CDN), Babel
